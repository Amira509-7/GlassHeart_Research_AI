# GlassHeart AI

GlassHeart AI is a machine learning project developed with my team to predict heart disease risk using multiple medical datasets.

The project combines data preprocessing, feature engineering, dataset merging, BMI calculation, missing value handling, and a Random Forest classification model to improve prediction accuracy.

Our goal was to build a reliable AI-based system that can analyze patient health data and assist in early heart disease detection.

## Features

* Multi-dataset integration
* Data cleaning and preprocessing
* BMI and health feature generation
* Missing data imputation
* Random Forest machine learning model
* Feature importance analysis

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## Team Project

This project was designed and developed collaboratively as part of our AI and data analysis learning journey.
